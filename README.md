#Protocol
Welcome to the Talky Protocol. It's so difficult, you might have a hard time learning it.

##Messages:

```
M:Message
```

##Commands:

```
M:/command args
```

##Ping (TBI)

```
P:PING
```

*Response:*

```
P:PING
```
